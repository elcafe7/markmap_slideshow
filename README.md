# Markmap Slideshow

A dynamic, web-based presentation tool that converts markdown-based **Markmap** mind maps into interactive, PowerPoint-style slideshows.

## 🚀 Overview

The **Markmap Slideshow** project bridges the gap between non-linear brainstorming (mind mapping) and linear presentations. It allows users to input standard markdown, renders it as a beautiful interactive mind map using the [Markmap](https://markmap.js.org/) library, and then seamlessly transitions those nodes into a full-screen presentation mode.

### Key Features (Planned/Implemented)
- **Markdown to Mind Map:** Rapidly draft complex structures using simple markdown text.
- **Dynamic Slideshow Mode:** Convert mind map nodes into sequential, easy-to-read slides.
- **Fullscreen Maximization:** One-click expansion for distraction-free presentations.
- **Local & Lightweight:** Operates entirely within the browser as a standalone HTML page. No complex backend or server required.

## 🛠️ Usage

*Note: The core HTML application is currently being staged for this repository.*

Once the `index.html` is available:
1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. Input your markdown in the provided editor.
4. Toggle between the **Mind Map** view and the **Slideshow** view to present your ideas.

## 💡 Why Markmap Slideshow?
Traditional presentation software can be slow for rapid idea generation. By combining the speed of markdown and mind-mapping with the structured delivery of a slideshow, this tool provides the best of both worlds for developers, educators, and planners.
